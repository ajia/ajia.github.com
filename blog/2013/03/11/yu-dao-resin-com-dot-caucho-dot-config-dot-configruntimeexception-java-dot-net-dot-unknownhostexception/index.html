遇到resin的java.net.UnknownHostException

linux启动resin4.0.23的时候，没有报错，但是应用也没有启动，访问的时候报com.caucho.config.ConfigRuntimeException: java.net.UnknownHostException，网上找了一下， 问题原因：DNS解析不到本机名。

解决方案：

编辑/etc/hosts,添加127.0.0.1 xxxx（本机名）

问题原因

resin启动时会根据hostname来拼url地址
