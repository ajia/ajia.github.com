Mar 6th, 2013

ruby -e "$(curl -fsSkL raw.github.com/mxcl/homebrew/go)"
brew update

Step2 安装RVM

bash -s stable < <(curl -s https://raw.github.com/wayneeseguin/rvm/master/binscripts/rvm-installer)
source ~/.bashrc

Step3 安装Ruby

rvm install 1.9.3
rvm use 1.9.3 --default
