学习ruby on Rails-ruby和rails的安装

by Ajia - Apr 1st, 2013 - posted in ruby | Comments

ruby的安装在文章how-to-install-octopress里有说过。

在Lion安装rails

gem install rails

创建rails的应用

rails new myapp

启动应用

cd myapp
rails server

进入

http://localhost:3000
